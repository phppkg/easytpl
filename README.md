⚡️ Simple and fastly text template engine for PHP

## Install

**composer**

```bash
composer require phppkg/fasttpl
```

## Usage

```php
use PhpPkg\FastTpl\FastTemplate;

$t = new FastTemplate();
$t->parse();
```
