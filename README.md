⚡️ Simple and fastly text template engine for PHP

## Install

**composer**

```bash
composer require phppkg/easytpl
```

## Usage

```php
use PhpPkg\EasyTpl\FastTemplate;

$t = new FastTemplate();
$t->parse();
```
